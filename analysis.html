Анализ временных рядов

Методы исследования характеристик временных рядов для эконометрического моделирования

Тесты на стационарность

Проверка временных рядов на стационарность с помощью тестов ADF, KPSS, Phillips-Perron для определения порядка интеграции и выбора подходящей модели.

Тесты на коинтеграцию

Выявление долгосрочных равновесных связей между нестационарными рядами с помощью теста Йохансена, теста Энгла-Грейнджера и анализа причинности по Грейнджеру.

Анализ сезонности

Выявление и количественная оценка сезонных компонентов в данных с помощью спектрального анализа и методов декомпозиции временных рядов.

В разработке

Анализ структурных изменений

Выявление структурных сдвигов и точек разрыва во временных рядах с использованием тестов Чоу, CUSUM и рекурсивных методов для корректного моделирования.

В разработке

Анализ волатильности

Исследование изменчивости и условной гетероскедастичности временных рядов с помощью тестов ARCH/GARCH для обнаружения кластеризации волатильности.

В разработке

Декомпозиция временных рядов

Разложение временных рядов на трендовую, сезонную и случайную компоненты для детального изучения структуры и динамики временного ряда.
